Key Rule Confirmations — Effective June 2, 2026 Webinar
✓
Modifier TH Confirmed — Antepartum & Postpartum
Confirmed
AMA confirmed Modifier TH (“Obstetrical treatment/services, antepartum or postpartum”) must be appended to all antepartum and postpartum E/M claims filed under the new framework. Modifier TH is NOT applied to labor management codes (59080–59083) or delivery codes — antepartum and postpartum E/M only.
✓
Same-Day Complexity Upgrade — Report Complex Only
Confirmed
When labor starts as straightforward (59080) and upgrades to complex on the same calendar day, report only the complex code (59081). Do not report both. Exception: if a separate provider group takes over care, each group may bill their own labor management code for that date.
✓
Planned Cesarean — No Labor Management Code
Confirmed
A scheduled or planned cesarean has no associated labor management code. Labor management codes apply only when labor management services are actually performed. Report delivery code only (59502 or 59503).
✓
Labor Management + Delivery — Billable Same Day
Confirmed
A delivery code may be reported on the same calendar date as an initial or subsequent labor management code, even when the same physician provides both. Explicitly confirmed — one of the most common same-day scenarios under the new framework.
✓
VBAC / TOLAC Always Complex
Confirmed
All TOLAC/VBAC patients are complex by default. “No previous cesarean delivery” is a required criterion for straightforward classification. VBAC patients fail that criterion automatically — use 59081 (initial) or 59083 (subsequent) regardless of other clinical factors.
✓
59504 — Hysterectomy Now Standalone (Not Add-On)
Confirmed
The old add-on code +59525 is deleted. It is replaced by 59504 as a standalone CPT code. May be reported by the same or a different physician who performed the cesarean delivery.
✓
Delivery Includes Same-Day Postpartum — Do Not Bill Separately
Confirmed
Routine postpartum care on the date of delivery is included in the delivery code valuation. Do not separately bill a postpartum E/M code on the same calendar date as delivery. Postpartum E/M begins with the first encounter after the delivery date.
✓
Primary C-Section: Same-Day Hospital E/M Separately Reportable
Confirmed
For a primary cesarean (59502), the initial hospital E/M (99221–99223) on the same date may be separately reported. For a repeat cesarean (59503), the admission E/M is included in valuation and cannot be billed separately.
✓
Pregnancy = “Problems Addressed” for MDM
Confirmed
AMA confirmed pregnancy is evaluated under the “Problems Addressed” element of Medical Decision Making. Complexity of the pregnancy drives the E/M level. Each prenatal note must independently support the level billed.

AMA hosts a national webinar specifically addressing the 2027 maternity care restructuring. Key confirmations: Modifier TH usage, same-day complexity upgrade rule, planned cesarean exclusion, labor+delivery same-day billing, VBAC complexity classification, postpartum-on-delivery-date bundling, and primary vs. repeat cesarean E/M rules.

CMS and major payer contractors expected to release Medicare/Medicaid billing guidance for the new codes, including national RVU values, LCD/NCD updates, and payer-specific Modifier TH acceptance policies. Monitor payer bulletins closely in Q3–Q4 2026.
Q4 2026 — Deadline
EHR Charge Capture & Billing System Updates Required
All EHR order sets, charge capture templates, fee schedules, and billing system code tables must be updated before December 31, 2026. Crossover pregnancy billing workflows must be implemented for patients spanning 2026–2027.
December 31, 2026
All Global / Antepartum Bundle Codes Expire
Final date of service under old codes. Any 2026 dates still pending must be billed under old framework. 2027 dates of service submitted with deleted codes will reject.
January 1, 2027
GO-LIVE — All New Codes Effective
New unbundled coding framework fully effective. Every OB/GYN service — antepartum, labor management, delivery, and postpartum — billed individually. No parallel reporting period. All teams must be trained and all systems updated before this date.

Required Practice Actions — Before January 1, 2027
⚠
Update EHR Charge Capture Templates
Action Required
Remove deleted global codes from all OB order sets and charge capture templates. Add new labor management codes (59080–59083), new delivery codes, and postpartum E/M workflows with Modifier TH. Target: complete by November 1, 2026.
⚠
Train Billing & Coding Staff
Action Required
All billers and coders must understand the new framework before go-live. Key topics: labor complexity determination, same-day billing rules, Modifier TH application, crossover pregnancy handling (2026 vs. 2027 rules by date of service).
⚠
Physician Documentation Education
Action Required
Physicians must document MDM elements in each prenatal note to support E/M code selection. Labor notes must document complexity criteria. Delivery notes must capture laceration degree. Education module under development.
▶
Crossover Pregnancy Billing Protocol
In Progress
Build workflow for pregnancies spanning 2026–2027. Antepartum visits in 2026 bill under old rules; visits in 2027 bill as individual E/M. Identify patients expected to deliver Q1 2027 and flag EHR records for dual-year billing.
▶
Payer Contract Review
In Progress
Review payer contracts for reimbursement language tied to deleted global codes. Initiate re-negotiation discussions where global code rates are referenced. Monitor payer bulletins Q3–Q4 2026 for updated fee schedules.
★
Revenue Impact Modeling
Planned
Model expected revenue change using new RVU values (once published by CMS). Compare against current global code reimbursement. Identify high-volume payers and service types to prioritize in contract negotiations.
