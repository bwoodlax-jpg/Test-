Connect a Data Source

Yahoo Finance blocks direct browser requests. Enter a free Finnhub API key to power the dashboard — takes about 60 seconds to get one.

  1. 1 Go to finnhub.io/register — free account, no credit card.
  2. 2 After signing in, your API key is shown on the dashboard. Copy it.
  3. 3 Paste it below. The key is saved in your browser — enter it once, done.

Free tier: 60 API calls/minute — more than enough for this dashboard.
Your key never leaves your browser (stored in localStorage only).
Running Flask locally? python app.py → data loads automatically via yfinance — no key needed.
